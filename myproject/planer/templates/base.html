Welcome to ePlan - your personal planner application.

With this app you can create lots of events and manage them. This will allow you to plan your time efficiently. As you know, time is money!

This application is study project and it will not be used in the near future for commercial purposes.

Usually, web applications consist of two layers - Frontend and Backend, and that's how it's in this app.

Frontend is based on CSS, HTML, JavaScript - jquery.

Backend - Python and it's popular framework, django.

Frontend

- HTML5

- CSS3

- JavaScript

- Jquery, JS's library.

Backend

- Python

- Django, Python's framework.

This application was created because of my interest in Web Development. I'm a self-taught man and I'm learning web programming for about a year. I'm most interested in Frontend programming, however, I'm trying to raise my Backend skills too. That's because in the future I would like to become a full-stack developer :)
